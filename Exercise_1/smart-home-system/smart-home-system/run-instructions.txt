Compile & Run (Java 11+):

# From the root of the unzipped project:
# 1. Compile java files
javac -d out $(find src/main/java -name "*.java")

# 2. Run the application
java -cp out com.smarthome.app.App

# Notes:
# - Use 'help' in the program for available commands.
# - To stop the app type 'exit'.
